Here's what we do (hopefully well)

We provide you with a simple admin panel that allows you to start as many server as you want (or your hardware can support), be it vanilla or modded Minecraft

One File to Rule Them All

The admin panel, the web interface and the underlying logic, all in a single file

All your favorite modpacks

We support downloading modpacks directly from Modrinth and CurseForge(TBD)

Your Data, Always

SimpleMinePanel never phones home (expect to check for updates, to be honest) and stores all your data locally

Sandboxed Environments

SimpleMinePanel runs each of the game servers inside a Docker container, so no contamination can ever spread to other game instances
